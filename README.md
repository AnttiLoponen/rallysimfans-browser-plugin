# rallysimfans-browser-plugin
Browser Plugin for www.rallysimfans.hu website

## Features

- Show differences in s/km in rally results and record times
- Hide undriven stages in record times or password protected rallies
- Filter rallies by text and pin favourite rallies to top of the list

## Installation

- Download files as zip and unarchive to a folder
- Browse to chrome://extensions/
- Select "Load unpacked"
- Select folder where you unarchived the files
- Browse to rallysimfans.hu or refresh and you should see the new features
